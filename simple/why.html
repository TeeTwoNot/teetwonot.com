This is what the web used to look like!

It might seem weird at first, after being used to browsing sites with amazing interactive graphics and smooth interfaces, but...

This was actually what most websites looked like back when HTML was on version 1!

The reason why there's a simple version of my website is because, in the rare event that someone is using really old hardware to access the web, they can visit this website really easily!

Some reasons that old hardware is capable of visiting this site are:

  • This website doesn't force HTTPS! Some older browsers unfortunately don't support it.
  • This is pure HTML! There is no CSS and no JavaScript. It's as simple as it gets.
  • It's nice and compact! Even a really old 56k dial-up connection can load this website pretty fast!
  • That's about it!

Now, even though simplicity is awesome, complexity is awesome too, so if you're not using Netscape to access this wesbite, you should visit teetwonot.com
